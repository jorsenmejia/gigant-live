Allow specify default placeholder image ( "NO IMAGE" ) for products by woocommerce.

== Description ==

Allows specify placeholder image ( "NO IMAGE" ) for Woocommerce-products.
Just follow to Woocommerce - Settings - Products - Display, select image, which you want to use as default.
Enjoy!
